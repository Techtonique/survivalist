Citing survivalist
======================

please cite the following paper:

    S. Pölsterl, "survivalist: A Library for Time-to-Event Analysis Built on Top of scikit-learn,"
    Journal of Machine Learning Research, vol. 21, no. 212, pp. 1–6, 2020.
